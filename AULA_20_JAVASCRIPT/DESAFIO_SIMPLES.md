
# 🚀 Desafio: Lista de Tarefas com LocalStorage

## 🎯 Objetivo

Criar uma aplicação simples de lista de tarefas usando:

* HTML
* CSS
* JavaScript

E salvar os dados no **localStorage**.

---

## 🧠 O que você precisa fazer

Sua aplicação deve permitir:

* ➕ Adicionar uma tarefa
* 📋 Listar as tarefas
* ❌ Remover uma tarefa

---

## 💾 Importante

* As tarefas devem ser salvas no **localStorage**
* Ao atualizar a página, elas **não podem desaparecer**

---

## 🖥️ Interface mínima

* Um campo de texto
* Um botão "Adicionar"
* Uma lista de tarefas

---

## 🧩 Exemplo de tarefa

```js id="exemplo1"
{
  texto: "Estudar JavaScript"
}
```

---

## ⚠️ Regras

* Não usar frameworks
* Usar apenas JavaScript puro
* Código organizado

---

## 🏁 Entrega

* `index.html`
* `style.css`
* `script.js`

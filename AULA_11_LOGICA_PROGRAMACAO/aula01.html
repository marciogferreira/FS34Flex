Lógica de Programação

e Algoritmos

Introdução para iniciantes

O que é Lógica de Programação?

  • Forma de pensar para resolver problemas
  • Organizar passos de maneira clara
  • Base para qualquer linguagem de programação

O que é um Algoritmo?

Um algoritmo é uma sequência de passos finitos para resolver um problema.


Exemplo:
1. Pegar ingredientes
2. Preparar a massa
3. Assar o bolo

Características de um Algoritmo

  • Entrada
  • Processamento
  • Saída

Entrada, Processamento e Saída


Entrada: dois números
Processamento: somar os números
Saída: resultado da soma

Tipos de Algoritmos

  • Descrição narrativa
  • Fluxograma
  • Pseudocódigo

Pseudocódigo


Início
  Leia A
  Leia B
  Soma ← A + B
  Escreva Soma
Fim

Estruturas Sequenciais

As instruções são executadas em ordem, de cima para baixo.


Leia nome
Leia idade
Escreva nome
Escreva idade

Estruturas Condicionais

Permitem tomar decisões.


Se idade >= 18 então
  Escreva "Maior de idade"
Senão
  Escreva "Menor de idade"
FimSe

Estruturas de Repetição

  • Enquanto
  • Para
  • Repita até

Exemplo de Repetição


Para i de 1 até 5 faça
  Escreva i
FimPara

Boas Práticas

  • Usar nomes claros
  • Comentar quando necessário
  • Manter o algoritmo simples

Exercício

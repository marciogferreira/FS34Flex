React Hooks

Entendendo o conceito e uso

O que são Hooks?

Funções que permitem usar recursos do React em componentes funcionais.

Por que usar Hooks?

  • Evita classes
  • Código mais simples
  • Reutilização de lógica

useState


const [contador, setContador] = useState(0);

Exemplo useState


function Contador() {
  const [contador, setContador] = useState(0);

  );

useEffect


useEffect(() => {
  console.log('Executou');
}, []);

Outros Hooks

  • useContext
  • useRef
  • useMemo
  • useCallback

Regras dos Hooks

  • Somente no topo do componente
  • Não usar dentro de loops ou condições
  • Somente em funções React

Hooks Personalizados


function useContador() {
  const [valor, setValor] = useState(0);

  function incrementar() {
    setValor(valor + 1);
  }

  return { valor, incrementar };
}

Resumo

Hooks simplificam o uso de estado e lógica no React moderno.
